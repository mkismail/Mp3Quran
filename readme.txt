== Description ==

Mp3 Quran Translations in English plugin is the first WordPress plugin that allows you to display a playlist for the mp3 files in English translations to understand the meaning of the Quran in english, powered by TrueMuslims.Net.

PS: For Support: [MP3 Quran)](http://www.truemuslims.net).

== Installation ==

* Upload the Mp3 Quran Translations plugin to wordpress site or blog, Activate it.
* Go to Mp3 Quran from menu.
* Copy yellow code like this: mp3_playlist[*] and past code in your post or page.
* Done!
